# ricardogs-econ.github.io

Academic portfolio site for Ricardo Gonçalves da Silva — researcher in Economics and Statistics, specializing in Econometrics, Time Series Analysis, and Applied Stochastic Processes.

Built with [Jekyll](https://jekyllrb.com/) using the [al-folio](https://github.com/alshedivat/al-folio) theme.

## Local development

```bash
bundle install
bundle exec jekyll serve
```

Site content lives in `_pages/` (About, CV, Publications), `_data/cv.yml` (structured CV data), and `_bibliography/papers.bib` (publications).
